BTech Daily Task Manager



A Task Manager is a simple web application that helps users organize, plan, and track their daily activities in an efficient way. It is especially useful for B.Tech students to manage their syllabus, assignments, lab work, and project tasks. By using a task manager, students can clearly list what they need to study each day and follow a structured schedule.
